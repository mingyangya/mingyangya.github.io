插件

  • vetur & vue-helper

    vue语言的支持，语法高亮

  • open in brower

    于浏览器中预览html网页

  • Live Server

    静态服务，实时预览

  • Import Cost

    计算npm包的大小

  • Image preview

    图片预览

  • GitLens

    查看Git相关信息

  • eslint

    js校验

  • Chinese (Simplified) Language Pack for Visual Studio Code

    适用于 VS Code 的中文（简体）语言包

  • Auto Close Tag

    自动闭合html标签

  • Auto Rename Tag

    自动检测配对标签，同步修改。

  • Bracket Pair Colorizer

    用不同的颜色来区分闭合括号

  • vscode-icons

    文件图标

  • Code Spell Checker

    检查单词拼写

快捷键

  • 编辑器与窗口

    ctrl+shift+p： 打开命令面板

    ctrl+p： 切换文件（输入文件名称）

    ctrl+j：切换终端显隐

  • 代码编辑

    ctrl+d：选中下一个相同内容

    ctrl+shift+l：选中所有相同内容

    ctrl+F：查找内容

    ctrl+shit+f：在整个文件夹中查找内容

    ctrl+shift+k：删除光标选择区域内的所有行
